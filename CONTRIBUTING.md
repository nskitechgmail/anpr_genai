# Contributing to Smart City ANPR System

Thank you for your interest in contributing! This project is an M.Tech research project
at SRM IST and welcomes community contributions.

## Getting Started

1. **Fork** the repository on GitHub
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/mtech_anpr_project.git`
3. **Create a branch**: `git checkout -b feature/your-feature-name`
4. **Make changes** and add tests
5. **Run tests**: `pytest tests/test_suite.py -v -m unit`
6. **Push** and open a **Pull Request**

## Development Setup

```bash
pip install -r requirements.txt
pip install pytest pytest-cov
```

## Code Style

- Follow PEP 8
- Add docstrings to all public classes and functions
- Maintain test coverage above 84%

## Reporting Bugs

Please open a GitHub Issue with:
- Python version and OS
- Steps to reproduce
- Expected vs actual behaviour
- Any error traceback

## Areas for Contribution

- Additional Indian state license plate format support
- Multi-camera synchronisation
- Improved safety classifier training data
- Jetson Nano / Raspberry Pi optimisation
- Dashboard UI improvements
